# creamy-chat

A peer-to-peer browser chat built as a learning project: Go signaling server + React/TypeScript frontend + WebRTC DataChannels for the actual messages. The goal is to see the networking concepts from *High Performance Browser Networking* live on the wire.

## Structure

- `server/` — Go signaling server (WebSocket)
- `web/` — React + TypeScript + Vite frontend
- `CLAUDE.md` — learning-mode instructions for Claude Code

## Prerequisites

- Go (latest stable)
- Node 20+
- GoLand (for the backend)
- VS Code (for the frontend)
- Wireshark (for observing the wire)
- `mkcert` (for local TLS — added in Phase 2)
